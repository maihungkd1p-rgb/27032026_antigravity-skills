# Hướng dẫn Format DOCX

Module này quy định cách trình bày nội dung để khi chuyển sang file Word (.docx) sẽ đảm bảo tính chuyên nghiệp và chuẩn mực.

## 1. Font & Typography
*   **Font**: Khuyên dùng **Times New Roman** (cho văn bản hành chính) hoặc **Arial/Calibri** (cho văn bản hiện đại).
*   **Size**: 12pt (Body text), 14pt (Heading 2), 16pt (Heading 1).
*   **Styles**:
    *   **Bold**: Dùng cho Heading và từ khóa quan trọng.
    *   *Italic*: Dùng cho chú thích, trích dẫn, tên tài liệu nước ngoài.

## 2. Heading Structure
Sử dụng phân cấp rõ ràng:
*   `# Heading 1` (Tiêu đề chính - Title)
*   `## Heading 2` (Mục lớn)
*   `### Heading 3` (Mục nhỏ)

## 3. Bảng Biểu (Tables)
Khi yêu cầu tạo bảng:
*   Luôn định nghĩa rõ Header row.
*   Căn lề: Text căn trái, Số căn phải, Tiêu đề căn giữa.
*   Tránh gộp ô (merge cells) quá phức tạp nếu không cần thiết.

## 4. Ngắt Trang (Page Breaks)
Nếu nội dung dài hoặc chuyển sang phần mới (ví dụ: Phụ lục), hãy thêm chỉ báo ngắt trang rõ ràng:
`--- PAGE BREAK ---`

## 5. Metadata
Luôn bao gồm thông tin Header/Footer nếu cần:
*   Top-left: Tên công ty / Logo
*   Top-right: "Tài liệu lưu hành nội bộ"
*   Bottom-center: Số trang

## Ví dụ Yêu Cầu
"Tạo báo cáo docs với font Times New Roman, cỡ chữ 12, tiêu đề đậm, có mục lục tự động..."
