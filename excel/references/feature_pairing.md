# Bảng Ghép Nối: Nhu Cầu Người Dùng → Tính Năng Excel

Đọc tài liệu này khi cần quyết định **tổ hợp tính năng** cho một file Excel cụ thể.
Không nhồi tất cả — chọn 3-5 tính năng phù hợp nhất với kịch bản.

---

## 1. Giúp người dùng 「Hiểu dữ liệu」

| Tính năng | Giá trị | Khi nào dùng |
|---|---|---|
| Biểu đồ cột/thanh | So sánh nhanh giữa các danh mục | So sánh giá trị |
| Biểu đồ đường | Nhìn xu hướng theo thời gian | Dữ liệu chuỗi thời gian |
| Biểu đồ tròn | Xem tỷ lệ phần-tổng thể | ≤6 danh mục |
| Data Bars (thanh trong ô) | So sánh độ lớn mà không rời ô | Cột số cần so sánh nhanh |
| Color Scale (thang màu) | Hiệu ứng heatmap, làm nổi bật patterns | Ma trận, phân phối |
| Sparklines (biểu đồ mini) | Xu hướng trong 1 ô | Hàng tóm tắt + lịch sử |

## 2. Giúp người dùng 「Tìm điều quan trọng」

| Tính năng | Giá trị | Khi nào dùng |
|---|---|---|
| Pre-sorting | Data quan trọng nhất lên đầu | Ranking, Top N |
| Conditional formatting | Dữ liệu chính tự nổi bật | Ngoại lệ, ngưỡng, Top/Bottom |
| Icon Sets | Trạng thái hiển thị nhanh | KPI (dùng tiết kiệm) |
| In đậm / Tô màu | Phân biệt chính / phụ | Hàng tổng, chỉ số chính |
| Phần KEY INSIGHTS | Trình bày kết luận trực tiếp | Báo cáo phân tích |

## 3. Giúp người dùng 「Tiết kiệm thời gian」

| Tính năng | Giá trị | Khi nào dùng |
|---|---|---|
| Overview Sheet | Tóm tắt trang đầu, không tìm kiếm | File ≥3 sheets |
| Tóm tắt tính sẵn | Kết quả sẵn sàng | Data cần thống kê |
| Number format nhất quán | Không cần điều chỉnh | Mọi dữ liệu số |
| Freeze Panes | Tiêu đề hiển thị khi cuộn | Bảng >10 dòng |
| Mục lục có hyperlink | Điều hướng nhanh | File ≥3 sheets |

## 4. Giúp người dùng 「Sử dụng trực tiếp」

| Tính năng | Giá trị | Khi nào dùng |
|---|---|---|
| Auto-filter | Tự khám phá dữ liệu | Bảng >20 dòng |
| Hyperlinks | Nhấp để chuyển, không đoán | Tham chiếu chéo sheets |
| Bố cục thân thiện in ấn | Sẵn sàng in/PDF | Báo cáo chia sẻ |
| Formulas (không hardcode) | Thay input → kết quả tự cập nhật | Mô hình, dự báo |
| Data Validation dropdown | Ngăn lỗi nhập liệu | Template cần user nhập |

## 5. Giúp người dùng 「Tin tưởng dữ liệu」

| Tính năng | Giá trị | Khi nào dùng |
|---|---|---|
| Ghi nguồn dữ liệu | Biết data từ đâu | Mọi data bên ngoài |
| Ngày tạo | Biết độ mới | Báo cáo nhạy thời gian |
| Phạm vi thời gian | Biết coverage | Chuỗi thời gian |
| Định dạng chuyên nghiệp | Trông đáng tin cậy | File hướng ra ngoài |
| Số chính xác nhất quán | Không nghi ngờ | Mọi giá trị số |

## 6. Giúp người dùng 「Thu được insight」

| Tính năng | Giá trị | Khi nào dùng |
|---|---|---|
| Cột so sánh (Δ, %) | Không tính thủ công | YoY, MoM, A vs B |
| Cột xếp hạng | Vị trí trực tiếp | Cạnh tranh, hiệu suất |
| Tóm tắt theo nhóm | Tổng hợp theo chiều | Phân tích phân đoạn |
| Chỉ báo xu hướng (↑↓) | Hướng rõ ràng | Thay đổi quan trọng |
| Văn bản insight | "Vậy thì sao?" nêu rõ | Báo cáo phân tích |
